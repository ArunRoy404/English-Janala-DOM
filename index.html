Frequently Asked Questions

1. The difference between var, let, and const?
var: a variable that can be re declared outside its scope.

let: a variable, which value can be changed inside its scope, can not be accessed outside its scope or block.

const: a variable, which value cannot be changed, and neither can be accessed outside its scope or block.
2. The difference between map(), forEach(), and filter() ?
map(): performs operation in each element of an array and returns the new array.

forEach(): performs operation in each element of an array but does not returns anything.

filter(): returns a new array of elements that passes a specific condition applied on an array.
3. Explain arrow functions and how they are different from regular functions.
Arrow function is defined using () => { } instead of function.

Arrow functions does not have an arguments object. It uses rest parameters instead.

Arrow functions is ideal callbacks, array operations, etc.
4. How JavaScript Promises work?
JavaScript promises represents the state and result of the fetched data.

It contains the state if the data is pending, available or rejected.
5. How closures work in JavaScript?
Closures is a way to access the properties and execute a specific operation through the inner function, when the outer function is called or finished execution at least once.

This inner function can be accessed from the outer scope of the outer function.
